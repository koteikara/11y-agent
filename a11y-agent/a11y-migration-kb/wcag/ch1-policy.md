# 配慮対象と内容
- 全盲：代替テキスト・正しいHTML構造でスクリーンリーダー対応
- 弱視：文字サイズ調整可・高コントラスト
- 色覚障がい：テキスト・記号を色と併用
- 聴覚障がい：音声情報に字幕・テキスト
- 上肢障がい：キーボードのみで全操作可
- 認知障がい：平易な表現・論理的構造
- 低速回線：軽量・標準的な実装
- 外国人：英語概要ページを用意
