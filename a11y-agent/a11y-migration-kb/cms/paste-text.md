# 説明
テキストは「プレーンテキスト貼り付け」で貼る（不要なHTML混入防止）。右クリック貼付け・Ctrl+V・通常貼付けは行わない。表は「ワード文書から貼り付け」後、貼付け箇所を選択して「書式を解除」で不要タグを除去する。Enterは段落、Shift+Enterは段落内改行。見出し設定は段落単位。
