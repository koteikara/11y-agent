# 説明
無操作（サーバ無通信）が30分続くと自動ログアウトする。中断時は必ず一時保存する。テキストエディタ編集が30分以上続くと更新が無いとみなされログアウトするため、長時間作業時もこまめに保存する。エディタでは必ず「編集終了」→ページ「一時保存」の順。プレビューで表示確認できる。
