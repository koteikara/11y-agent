# 説明
機種依存文字は特定環境でしか正しく表示されない文字。入力時・更新/承認申請時にチェックがかかり、「変換」で代替文字に自動変換できる。禁則文字（管理者登録のNGワード）がある状態では更新・承認申請できない。外字など変換候補が「？」になる場合は質問する。
