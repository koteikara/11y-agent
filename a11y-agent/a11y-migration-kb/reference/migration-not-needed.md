# 説明
以下は原則移行不要（自治体別指示があればそれに従う）。
- 更新日（移行が必要な場合は自治体別マニュアル）
- Adobe Reader等のファイル閲覧ソフトのDLリンク（PDF掲載時にCMSが自動掲出）
- 用語解説（Weblio辞書サイトへのリンク）
- お問い合わせ欄（ページの署名。設定済み）

# 質問が必要なケース（お問い合わせ欄）
署名が未設定／指定と別の署名／複数署名／名前は正しいが電話・FAXが違う、の場合は質問する。
