# 移行作業欄
- 移行開始日 / 移行担当者 / 作業者メモ
- 質問欄(1)〜(3)（SV宛て）/ 回答欄(SVより)（SV登録）/ 回答欄(作業者より)（作業者間共有用）
- (最終)移行担当者 / 移行完了日
- h1見出し修正：ページタイトル変更時「（元タイトル）」→「（新タイトル）」
- 報告欄（自治体・SVへの報告。リンク切れ/ファイル切れ以外）
- リンク切れ・リンク名・URL：「（リンク名）」（URL）形式
- ファイル切れ・ファイル名・URL：「（ファイル名/画像名）」（URL）形式

# 公開作業欄
- 公開作業担当者 / 公開作業開始日
- 差戻し内容（スタッフ宛て。修正5分以上のとき）/ 修正担当者 / 修正日
- 質問欄・回答欄（公開作業内）
- (最終)公開作業担当者 / 公開作業完了日
- 公開せずに完了：移行のみ完了させ公開しない場合
- SV指示で中断：作業途中で移行不要になった場合（1ページとしてカウント）
