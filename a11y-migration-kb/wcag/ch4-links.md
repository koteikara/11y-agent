# 4.1 画像リンクのalt [JIS:2.4.4]
リンク画像のaltにリンク先が明確に分かる言葉を設定。

# 4.2 新規ウィンドウの事前通知
target="_blank"はリンクテキストまたはアイコンで事前通知。

# 4.3 リンクテキスト [JIS:2.4.4]
リンク単体で内容が理解できる表現。禁止: 「こちら」「ここ」「詳細」等のあいまい表現のみ。
