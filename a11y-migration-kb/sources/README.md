# sources — 原本マニュアル・一次資料の保管場所

このディレクトリは、`a11y-migration-kb` の各ルールの根拠となる**原本マニュアル・一次資料（PDF等）**を保管する場所です。

## 目的
- KBルールの frontmatter にある `resource: manual://V2.01#p61` のような参照が、実際にどの資料のどのページを指すのかを追跡できるようにする。
- ルール文言と原本の差分レビュー（原本に沿ってKBを補強する作業）の元資料を、リポジトリ内に一元管理する。

## 置き方（ファイル名の目安）
- アクセシビリティ改修マニュアル本体: `manual-v2.01.pdf`（版数はファイル名に含める）
- 版が上がった場合は旧版も残し、`manual-v2.01.pdf` / `manual-v2.02.pdf` のように併存させる。

## 対応表（原本 → KBルール）
原本のどのページがどのルールに対応するかは、各ルールの frontmatter `resource` フィールドで管理します。主なもの:

| 原本参照 | 対応KBルール |
|---|---|
| `manual://V2.01#p61` | `rules/image/alt-text.md`（画像名の考え方＝主題＋様子＋+α情報） |
| `manual://V2.01#p63` | `rules/image/complex-image-report.md`（複雑な画像の補足説明・報告） |
| `manual://V2.01#p64` | `rules/image/caption.md`（画像のキャプション） |

> 注: 原本PDFは配布・取り扱いポリシーに従って追加してください。機微情報を含む資料は取り扱いに注意します。
