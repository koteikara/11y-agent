# 説明
あらかじめ入力エリアが用意されたテンプレートと、自由に組み合わせる0005:一般ページ用（自由レイアウト型）がある。基本は0005を使用。指定がある場合は指示に従う。テンプレートを変更すると入力データは削除される（移行元データのみ保持）。
