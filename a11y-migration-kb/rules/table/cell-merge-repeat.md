# 必須ルール
結合されたセルで同じ項目（同じ値）を複数行・複数列にまたがって表現している場合は、読み上げで対応関係が崩れるため、次のいずれかで整理する。

- **⑦ セルを分割して同じ項目を記載する**: 結合をやめ、各行・各列に同じ値をそれぞれ書く。
- **⑧ 表の下に記載する**: 全行に共通する補足（限度額・条件など）は、表内の結合セルではなく表の下の本文に書く。
- **⑨ 2つの表に分割する**: 結合セルが実質的に別の観点をまとめている場合は、観点ごとに2つの表へ分割する。

どの方法にするかは、表の内容と読み上げのしやすさに応じて判断する。

# 例
## ケース1: 同じ「備考」を結合で繰り返している
```before
<td rowspan="3">親子（準備：水筒・上履き）</td>
```
```after
各行の「備考」に「親子（準備：水筒・上履き）」をそれぞれ記載する（⑦セル分割）。全行共通の注意であれば、表の下の本文に記載する（⑧表の下に記載）。
```
ポイント: 結合で「同じ項目だから省略」した状態は、読み上げでどの行に対応するかが伝わらない。分割して各行に記載するか、共通事項は表の外へ出す。
