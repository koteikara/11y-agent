# ポイント
- 作業前に移行元ページをざっと確認し、不明点は早めに質問する（自治体確認が必要な質問は回答に時間がかかる）。
- 回答欄(作業者より)を使い、SV回答前でも作業者間で情報共有する。
- 質問は丁寧に詳しく。SV担当はページ内容を全把握していないため、該当箇所・内容を素早く把握できるよう具体的に書く（「リンク切れがあります」だけでは特定に時間がかかる）。
